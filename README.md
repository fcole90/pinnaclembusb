# Pinnacle MovieBox USB Linux driver

A Linux user-space library for the Pinnacle MovieBox USB video capture/playback device

## Requirements

- `libusb-dev`

On ubuntu

```sh
sudo apt install libusb-dev
```

## Compile

```sh
make all
```

## Run

```sh
 ./bin/pmbpipe
```

```sh
./bin/pmbplay FILE
```
